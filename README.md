# Análise de Sistema Real - Tropykaly Pizzas e Lanches

**Aluno:** João Gabriel Oliveira Barbosa
**Disciplina:** Arquitetura de Sistemas
**Curso:** Análise e Desenvolvimento de Sistemas

## Objetivo da Atividade
Esta atividade tem como objetivo aplicar conceitos de Arquitetura de Software, Design de Software e Engenharia Reversa baseada no comportamento de um sistema real em produção (Tropykaly Pizzas e Lanches), sem acesso ao código-fonte. A análise inclui modelagem UML e comparação com sistemas didáticos.
